# hexo-blog

#### install

* install hexo

```bash
npm install hexo --save
```

* verify success

```bash
hexo -v
```

#### start use hexo

* init

```bash
hexo init
```

* install dependents module

```bash
npm install
```

* create ports file

```bash
hexo new [layout] <title>
hexo n "filename"
```

* generate static html files

```bash
hexo generate
hexo g
```

* start server

```bash
hexo server
hexo s
```

* install images plugins

```bash
npm install hexo-asset-image --save
```
