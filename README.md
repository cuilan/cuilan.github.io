# hexo-blog

#### install

* install hexo

```shell
npm install hexo --save
```

* verify success

```shell
hexo -v
```

#### start use hexo

* init

```shell
hexo init
```

* install dependents module

```shell
npm install
```

* create ports file

```shell
hexo new [layout] <title>
hexo n "filename"
```

* generate static html files

```shell
hexo generate
hexo g
```

* start server

```shell
hexo server
hexo s
```

* install images plugins

```shell
npm install hexo-asset-image --save
```
