# hexo-blog

个人博客

---

## Hexo

#### Install

```bash
npm install hexo-cli -g
```

* verify success

```bash
hexo -v
```

#### Init

```bash
hexo init blog
npm install
```

#### Start hexo

```bash
hexo server
hexo s
```

#### Generate

```bash
hexo clean
hexo generate
hexo g
```

* create ports file

```bash
hexo new [layout] <title>
hexo n "filename"
```
