## Hexo

### Command

```bash
# Install
npm install hexo-cli -g

# verify success
hexo -v

# Init
hexo init blog
npm install

# Start hexo
hexo server
hexo s

# Generate
hexo clean
hexo generate
hexo g

# create ports file
hexo new [layout] <title>
hexo n "filename"
```
